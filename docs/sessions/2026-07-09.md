## Funcionalidades Implementadas

### 1. Sistema de Syntax Highlighting
- **Linguagens suportadas**: JavaScript, Python, Markdown, JSON, C/C++, Rust, PowerShell, Batch
- **Abordagem inicial**: Regex replace sequencial (quebrado)
- **Abordagem final**: Sistema token-based com deduplicação de overlaps
- **Tempo**: ~45 minutos (incluindo debug)

#### Problema Crítico Resolvido
**Sintoma**: HTML aparecia como texto visível (ex: `class="hl-string">`)

**Causa**: Quando múltiplos padrões regex eram aplicados sequencialmente, padrões subsequentes matchavam dentro do HTML gerado por padrões anteriores.

**Solução**: Sistema de tokens que:
1. Coleta todos os matches de todos os padrões
2. Ordena por posição e tamanho
3. Remove overlaps (primeiro/maior match vence)
4. Build HTML em uma única passagem

### 2. Suporte a PowerShell e Batch
Adicionada syntax highlighting completa para:

**PowerShell (.ps1)**:
- Comentários: `#` e `<# ... #>`
- Variáveis: `$variable`
- Cmdlets: `Get-*`, `Set-*`, `New-*`, etc.
- Operadores: `-eq`, `-ne`, `-like`, etc.

**Batch (.bat/.cmd)**:
- Comentários: `::` e `REM`
- Variáveis: `%var%` e `%%i`
- Labels: `:label`
- Keywords: `echo`, `if`, `for`, `goto`, etc.

### 3. Sistema de Execução de Scripts
- **Suporte**: .bat, .cmd, .ps1, .py, .sh, .exe
- **Controle**: Botão de parar com `taskkill /F /T /PID`
- **Segurança**: Validação de paths e extensões
- **Timeout**: 30 segundos automático
- **Auto-save**: Salva arquivo modificado antes de executar

#### Bug Corrigido
**Problema**: Scripts modificados executavam versão antiga do disco

**Causa**: `executeCurrent()` só salvava se `!tab.path`, ignorando `tab.dirty`

**Solução**:
```javascript
if (!tab.path || tab.dirty) {
  saveCurrentContent();
  await saveFile();
}
```

### 5. Drag-and-Drop de Abas
- **API**: HTML5 Drag and Drop
- **Visual**: Indicadores azuis de drop position
- **Bug corrigido**: Ícone de "proibido" ao arrastar sobre container vazio

**Solução**: Adicionar `dragover` handler no container:
```javascript
ct.addEventListener('dragover', (e) => {
  e.preventDefault();
  e.dataTransfer.dropEffect = 'move';
});
```

## Decisões Técnicas

### Por que Token-Based Highlighting?
- **Regex sequencial**: Simples mas quebrado (sobreposição de HTML)
- **Token-based**: Mais complexo mas robusto
- **Trade-off**: Performance vs. Correção (escolhemos correção)

### Por que 4 Espaços ao Invés de 2?
- **Padrão da indústria**: Python usa 4 espaços (PEP 8)
- **Legibilidade**: Mais fácil ver indentação
- **Consistência**: Alinha com maioria dos editores modernos

### Por que Inline JavaScript?
- **Simplicidade**: Single-file app, fácil de distribuir
- **Performance**: Sem network requests para carregar JS
- **Manutenibilidade**: Tudo em um lugar (para projeto pequeno)

## Bugs Encontrados e Corrigidos

### 1. Highlighting Corrompido
- **Sintoma**: `print(class="hl-string">"epic")`
- **Causa**: Regex matchando dentro de HTML gerado
- **Solução**: Sistema de tokens
- **Tempo de debug**: ~20 minutos

### 2. Drag-and-Drop Quebrado
- **Sintoma**: Ícone de "proibido" ao arrastar
- **Causa**: Falta `dragover` handler no container
- **Solução**: Adicionar handler no container pai
- **Tempo de debug**: ~5 minutos

### 3. Scripts Modificados Executando Versão Antiga
- **Sintoma**: Alterações não refletiam na execução
- **Causa**: Auto-save só funcionava para arquivos novos
- **Solução**: Verificar `tab.dirty` além de `!tab.path`
- **Tempo de debug**: ~10 minutos

### 4. Tab Não Funcionava
- **Sintoma**: Tab movia foco ao invés de inserir espaços
- **Causa**: Falta `preventDefault()` no keydown
- **Solução**: Adicionar handler de keydown com preventDefault
- **Tempo de debug**: ~3 minutos

### 5. Highlighting Não Aplicava em Arquivos Não Salvos
- **Sintoma**: PowerShell/Batch sem highlighting em novas abas
- **Causa**: `detectLanguage()` só usava `tab.path`, não `tab.title`
- **Solução**: Usar `tab.path || tab.title`
- **Tempo de debug**: ~8 minutos

## Aprendizados

### Técnico
1. **Regex em HTML**: Nunca aplicar regex sequencialmente em HTML já processado
2. **Drag-and-Drop**: Container precisa de `dragover` handler, não apenas elementos
3. **Auto-save**: Verificar dirty state, não apenas existência de path
4. **Tab em textarea**: Sempre `preventDefault()` no keydown
